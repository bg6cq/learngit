# 大家一起学 Git

> 🎯 **目标**：2 小时掌握 Git 和 GitHub 核心技能  
> 👥 **适合人群**：编程初学者、需要协作开发的团队  
> 📝 **前置要求**：基本的命令行操作知识

「工欲善其事，必先利其器」，Git 是开发者不可或缺的工具。本教程通过 8 节实践课，带你从零开始掌握 Git 版本控制和 GitHub 协作流程。

## 📚 课程目录

| 课次 | 主题 | 预计时间 | 核心技能 |
|------|------|----------|----------|
| [第一课](1/README.md) | 环境准备 | 10 分钟 | 安装 Git、注册 GitHub |
| [第二课](2/README.md) | 环境配置 | 10 分钟 | 配置用户信息、SSH 密钥 |
| [第三课](3/README.md) | 新建项目，跟踪修改 | 10 分钟 | init、add、commit、push |
| [第四课](4/README.md) | Fork 与 Pull Request | 30 分钟 | 协作开发核心流程 |
| [第五课](5/README.md) | 交流与沟通 | 5 分钟 | Issues、Stars、社区互动 |
| [第六课](6/README.md) | 分支管理 | 30 分钟 | branch、merge、rebase |
| [第七课](7/README.md) | 整理提交历史 | 15 分钟 | 交互式 rebase、commit 合并 |
| [第八课](8/README.md) | GitHub Pages | 10 分钟 | 搭建个人主页 |

**总计**：约 2 小时
